REDIRECTION
  • Method of redirecting data between files or programs
  • Output of a program can be saved in another file using the > operator
  • Input of a program can be read from a file using < operator
  • Eg: ls > list.txt will save the output of ls in a file named list.txt
  • >> can be used to append the output to a file while the > operator overwrites the data present in the file

PIPING
  • output go the program on the left is taken as the input to the program on the right
  • operator used is | (pipe)
